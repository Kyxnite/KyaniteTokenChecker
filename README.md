# Kyanite Token Checker
One of the **Fastest** token checkers coded **FULLY** in **GOLANG**



## Features

- **Multiple Tokens**
- **Multi-Threaded**
- **Token Check**
    - Username
    - Discriminator
- **Speed**
    - Extremely Fast
    - Multiple Token Files

## Installation

```go
go mod init
go mod tidy
```
## Usage/Examples


```go 
go run main.go
```
